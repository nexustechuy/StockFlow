# StockFlow oficial documentation
# This document includes a general description of the system and the update log

[En]
StockFlow is a web-based inventory management system designed to streamline stock control in retail stores. It enables administrators and stock clerks to monitor inventory, manage product replenishment, track stock levels in real time, and improve operational efficiency through a centralized and user-friendly platform.

[Es]
StockFlow es un sistema de gestión de inventarios basado en la web, diseñado para agilizar el control de existencias en tiendas minoristas. Permite a los administradores y al personal de almacén supervisar el inventario, gestionar el reabastecimiento de productos, monitorear los niveles de stock en tiempo real y mejorar la eficiencia operativa mediante una plataforma centralizada e intuitiva.
